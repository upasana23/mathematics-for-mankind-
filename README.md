📐 Mathematics for Mankind
Mathematics for Mankind is a high-end, full-stack educational platform designed to bridge the gap between YouTube learning and structured classroom success. Built for students from Class 6 to 12, the platform features a sleek "Deep Navy & Purple" aesthetic, smooth Framer Motion animations, and a robust system for academic resource management.

✨ Key Features
🎓 Student Portal
Class-Specific Hubs: Tailored notes and video links for Classes 6 through 12.

Vedic Maths & History: Dedicated sections for specialized mathematical shortcuts and historical context.

The Doubt Dump: A real-time interface where students can "dump" their mathematical roadblocks via text or image uploads from their gallery.

👨‍🏫 Teacher (Admin) Dashboard
Secure Access: Role-based authentication requiring a master ADMIN_SECRET_KEY.

Resource Management: Direct upload of PDF notes and solution images from the mobile gallery to the cloud.

Doubt Resolution: A private workstation for the teacher to view pending student queries and post handwritten solutions.

🎨 Premium UI/UX
Glassmorphism: Modern translucent components with glowing borders.

Animated Transitions: Staggered list reveals and "Reveal on Scroll" effects using Framer Motion.

Mobile Optimized: Designed specifically for easy use on mobile devices for both students and the teacher.

🚀 Tech Stack
Frontend: React.js, Tailwind CSS, Framer Motion, Lucide React.

Backend: Node.js, Express.js.

Database: MongoDB Atlas (Mongoose ODM).

Cloud Storage: Cloudinary (for PDF/Image hosting).

Authentication: JWT (JSON Web Tokens) with Bcrypt password hashing.
